Welcome.

Pick a warehouse, select a share and a BDC recipient, and publish. The dropdowns are filtered to what you can see in Unity Catalog (by direct grant or ownership). The GRANT SELECT ON SHARE runs as you via OBO so UC enforces ownership; the BDC SDK calls then run as the app's service principal on serverless — no workspace-admin role required. See the FAQ for the full identity model.
